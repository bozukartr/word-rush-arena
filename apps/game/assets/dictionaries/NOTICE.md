# Turkish word list attribution

`tr_words.txt` is derived from the Hunspell Turkish dictionary maintained by
the Turkish Data Depository (TDD) group:

- Source: https://github.com/tdd-ai/hunspell-tr (mirrored at
  https://github.com/LibreOffice/dictionaries/tree/master/tr_TR)
- License: Mozilla Public License 2.0 (see `LICENSE-MPL-2.0.txt`)

The file lists only the dictionary root forms (Hunspell affix flags stripped);
it does not include the full set of inflected word forms the original
Hunspell `.dic`/`.aff` pair can generate. This is a simplification, not a
license requirement — see `docs/DEPLOY.md` for the trade-off.
